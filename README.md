# 📝 Quick Notes

A modern and responsive note-taking web application built with **HTML**, **CSS**, and **JavaScript**. Quick Notes allows users to create, edit, delete, search, and manage notes directly in the browser with persistent storage using Local Storage.

## 🚀 Live Demo

<i><a href="https://yousuf-quicknote.netlify.app/">Project Link</a></i>

---

## ✨ Features

* Create new notes
* Edit existing notes
* Delete notes
* Search notes in real-time
* Dark / Light theme toggle
* Theme preference persistence
* Local Storage support
* Responsive design for mobile and desktop
* Keyboard shortcut support (`Ctrl + N`)
* Secure rendering with HTML escaping
* Empty state UI for better user experience

## 💡 How It Works

### Creating Notes

Users can create notes using a modal form. Notes are stored locally in the browser.

### Editing Notes

Existing notes can be updated without creating duplicates.

### Searching Notes

The search feature filters notes by title and content in real time.

### Theme Persistence

Selected theme preferences are saved in Local Storage and restored automatically on page reload.

### Data Persistence

All notes are saved using the browser's Local Storage API, ensuring data remains available even after closing the browser.
